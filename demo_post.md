# 【生活娱乐赛道】寄信——把说不出口的话，寄往未来的某个时刻

> 💌 有些话，值得被郑重地寄出，在对的时间抵达对的人手里。

---

## 一、Demo 简介

**是什么？**

「寄信」是一个极简的**定时信件投递 Web 应用**。写一封信、设定送达时间，到点自动发送到对方邮箱；也可以不填邮箱，生成二维码当面赠送。

**面向谁？**

愿意为情感表达多花一点心思的年轻人——情侣、好友、毕业生、给未来自己写信的人，以及不擅长当面表达、更习惯用文字传递心意的人。

**主要功能：**

| 功能 | 说明 |
|------|------|
| ✍️ 写信 | 简洁的写信界面，支持选择四套信纸主题 |
| ⏰ 定时投递 | 设定未来送达时间，后端自动轮询投递 |
| 📧 邮箱送达 | 到点自动发送邮件到收件人邮箱 |
| 🔗 二维码分享 | 不填邮箱也能生成二维码，当面赠送 |
| 🎬 打字机动画 | 收信人打开后，信件逐字展开，仪式感满满 |
| 🕐 阅后即焚 | 30 天后信件自动销毁，像真实信件一样有时效 |

## 二、Demo 创作思路

### 💡 灵感来源

现在的即时通讯太"快"了，消息发出去几秒就被刷走，反而让有分量的话显得廉价。我想做一个"慢"下来的表达方式，让一封信郑重地"寄出"，在对的时间抵达。

### 😣 想解决的问题

- 生日的祝福、毕业的不舍、想对未来的自己说的一句话，常常憋在心里、最后不了了之
- 即时通讯里写好的话发出去太随意，没有"被珍重对待"的感觉
- 想定时提醒又得依赖备忘录，容易忘
- 想保留仪式感只能手写实体信，成本高又难寄达

### 🎯 为什么做这个方向

在被即时消息填满的当下，人们反而更渴望"郑重表达"的仪式感。一个轻量、有温度、能精准定时送达的工具，正好填补了这块空白。
